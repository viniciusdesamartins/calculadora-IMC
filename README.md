# Calculadora IMC

Aplicação simples para calcular o Índice de Massa Corporal (IMC) a partir do peso e da altura informados pelo usuário.

# Funcionalidades

Entrada de peso (Kg) e altura (m)

Cálculo automático do IMC

Interface responsiva e moderna
